Ext.ux.HTMLEditorImage Plugin [Demo List]

This plugin provides a facility to insert and edit images in the Ext.ux.HTMLEditor (which is my extended version of the HTMLEditor). See HtmlEditor & Toolbar Extended for Extensibility for more information on this.

Also included is an image browser that has been refined from Jack's DataView example. See Jack's original version here: DataView Example.

All files that make up the full working version are included here, however I have disabled the Upload and Delete functions as I don't want people fiddling with this demo. All js files are verbose and commented.
